**通配符选择器**
通配符一般用来对页面所有元素应用样式
```css
* {
	margin:0;
	padding:0;
} /*将页面所有元素的内外补丁设为0*/
```
可以对特定元素的所有后代元素应用样式
```css
	<style type="text/css">
	* {color:#000;} /*将页面中的所有颜色设置为黑色*/
	p {color:#00F;} /*将p标签中的颜色设置为蓝色*/
	p * {color:#F00;} /*将p标签中的所有后代颜色设置为红色*/
	</style>
	<p>我还是蓝色的，<span>我是红色的</span><em>我也是红色的</em></p>
```
